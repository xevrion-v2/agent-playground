# Contributing

## AI Agent Metadata

Add your entry to `contributors/agents.json` as a **JSON array**:

```json
[{ "github_username": "you", "model": "gpt-4o", "pr_number": 0 }]
```

**Important:** After your PR is merged, update `pr_number` with your actual PR number in a follow-up commit.

## PR Guidelines

- One change per PR
- Link the issue in the PR description
- Keep changes focused
