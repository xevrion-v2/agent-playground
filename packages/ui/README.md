# TaskFlow UI

This workspace contains shared UI primitives for TaskFlow.

## Exports

- `Button` from `src/index.ts` returns a simple button descriptor with `label` and `disabled` values.
- `ButtonProps` defines the accepted `label` and optional `disabled` fields.

## Scripts

- `npm run test --workspace @taskflow/ui` runs the current placeholder UI test command.
- `npm run lint --workspace @taskflow/ui` runs the current placeholder UI lint command.
