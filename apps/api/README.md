# TaskFlow API

Express API service for the TaskFlow monorepo.

## Local Scripts

```bash
npm run dev --workspace @taskflow/api
npm run test --workspace @taskflow/api
```

## Current Stub Endpoints

- `GET /health` returns the API health status.
- `GET /users` returns an empty user list stub.
- `POST /users` returns a stub user object based on the request body.

These routes are placeholders for future controller, service, and validation layers.
