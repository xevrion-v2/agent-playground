# Engineering Playground — Challenge Index

## PI Calculation

**File**: `pi.py`

Calculates π using the Leibniz-Gregory series (Madhava variant).
Simple, dependency-free, and easy to run.

```bash
# Default (1M terms — ~5 decimal places)
python3 apps/api/src/challenges/pi.py

# Higher precision (10M terms — ~7 decimal places)
python3 apps/api/src/challenges/pi.py 10000000
```

See `pi.py` for implementation details and convergence notes.
