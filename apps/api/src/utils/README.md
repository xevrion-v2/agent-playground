# Utils

This directory contains utility functions used by the API.

## PI Calculation

This module calculates PI using the Leibniz series method for its implementation of calculating π (pi). It provides a lightweight algorithm approach to approximating the mathematical constant.

The calculation is done using the Leibniz formula: π = 4/1 - 4/3 + 4/5 - 4/7 + 4/9 - 4/11 + 4/13 - ...

For more information about the Leibniz formula, see [Leibniz formula for π](https://en.wikipedia.org/wiki/Leibniz_formula_for_%CF%80).
