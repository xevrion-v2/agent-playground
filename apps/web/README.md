# TaskFlow Web

This workspace contains the Next.js web application for TaskFlow.

## Entry Point

- `src/app/page.tsx` renders the current landing page content.

## Scripts

- `npm run dev --workspace @taskflow/web` starts the local Next.js development server.
- `npm run test --workspace @taskflow/web` runs the current placeholder web test command.
- `npm run lint --workspace @taskflow/web` runs the configured Next.js lint command.
