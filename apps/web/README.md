# Web App

The main Next.js application — the user-facing frontend.

## Entry point

`src/app/page.tsx` — root page rendered at `/`.

## Scripts

| Script  | Description                      |
| ------- | -------------------------------- |
| `dev`   | Start Next.js development server |
| `build` | Production build                 |
| `start` | Start production server          |
| `lint`  | Run ESLint across the app        |

## Conventions

- Uses Next.js App Router (`src/app/`)
- Route groups and layouts follow standard Next.js conventions
- API routes live in `src/app/api/` (if any)
