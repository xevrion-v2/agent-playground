# TaskFlow Web (`apps/web`)

Next.js 14 App Router frontend for the TaskFlow monorepo.

## Entry point

- `src/app/page.tsx` — landing page with the TaskFlow heading and workspace copy

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start the Next.js dev server |
| `npm run test` | Placeholder (no web tests configured yet) |
| `npm run lint` | Run `next lint` |

## Run locally

```bash
npm install
npm run dev -w apps/web
```
