# Bug Bounty Program

This repository participates in an open bug bounty program. Contributors are encouraged to find and report bugs, security issues, and improvement opportunities.

## How to Participate

1. **Find a bug** or improvement opportunity in the codebase
2. **Search existing issues** to avoid duplicates
3. **Open a new issue** describing the problem clearly
4. **Include /bounty $[amount]** in your issue body with a suggested payout
5. **Reference this issue (#33)** in your new issue
6. **Submit a pull request** with your fix
7. **Link the issue** in your PR description using `Fixes #N`

## Bounty Amounts

Bounties are posted with suggested amounts. Typical ranges:
- Documentation improvements: $10-$50
- Bug fixes: $25-$100
- Feature implementations: $50-$250
- Security vulnerabilities: $100-$1000

## Guidelines

- Keep pull requests focused on a single change
- Include a short summary of the change in the PR description
- Reference the related issue using `Fixes #N`
- Follow the project's coding standards and style

## Payment

Bounty payments are made via BTC or ETH. Include your wallet address in your PR description.

Happy hunting!
